Mode d’emploi express

Objectif : vérifier les créneaux, corriger ce qui cloche, puis partager une version de travail claire.
  1. Par club : vérifier ses créneaux dans le Planning des demandes ou dans le Scénario retenu.
  2. Modifier : corriger horaire, équipement, catégorie ou usage.
  3. Ajouter un créneau : signaler une demande manquante.
  4. Scénarios : construire une version de travail, la nommer, indiquer l’auteur et la date.
  5. Exporter le scénario : garder une copie de sécurité.
  6. Importer un scénario : récupérer une copie perdue ou reçue.
  7. Publier comme scénario retenu : envoyer la version partagée dans Google Sheet.
  8. Scénario retenu : consulter la version courante et les anciennes versions publiées.
Les scénarios locaux restent dans ce navigateur. Pour partager avec tout le monde, il faut publier comme scénario retenu.
