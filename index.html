Mode d’emploi clubs

Ce support sert à préparer le travail sur les créneaux 2026-2027. Les données peuvent être corrigées ou complétées pour faciliter les échanges.
  1. Aller dans Par club pour vérifier les créneaux de son association.
  2. Vérifier les horaires, équipements, catégories et usages.
  3. Utiliser Modifier pour corriger une information.
  4. Utiliser Ajouter un créneau si une demande manque.
  5. Choisir le bon statut : Existant à conserver, Demande en plus, À libérer, À déplacer / échanger ou À préciser.
  6. Utiliser Scénarios pour proposer une version de travail, depuis le Planning des demandes ou depuis un scénario vide.
  7. Donner un nom clair au scénario, indiquer l’auteur / club et l’objectif.
  8. Enregistrer le scénario localement.
  9. Utiliser Exporter le scénario pour garder une copie, puis Importer un scénario pour la récupérer si besoin.
  10. Imprimer ou exporter si besoin.
  11. Signaler simplement si quelque chose ne fonctionne pas ou si un créneau manque.
Les scénarios sont sauvegardés localement dans le navigateur. Pour transmettre une proposition, utilisez l’impression, Exporter le scénario ou envoyez une capture / PDF.
